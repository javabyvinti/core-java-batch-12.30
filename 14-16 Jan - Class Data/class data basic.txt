--> Keywords

	--> Resereved or predefined words
	--> int, char, break, if, switch, do, while, etc.


--> variables

two types:
	varying variables:
	value can be changed either at compile time or at run time.

	eg:
		int value = 10;
		String name = "Ram";

	--> datatype variable_name = initialization;	
	--> has a type and a value;

--> Fixed type variable:
	value cannot be changed neither at compile time nor at runtime

	final int value = 10;

	value = value + 10;	//error
